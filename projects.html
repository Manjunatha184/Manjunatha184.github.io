Client Relationship Management System (Developed with PHP, HTML, CSS, and JS on XAMPP.)

Real-Time Gesture Recognition System (Developed using Python, AIML, Image Processing.)

Diet and Workout Recommendation System (Developed using Python and AIML, based on BMI.)

I developed an intelligent system using Python and AIML to deliver personalized diet and exercise plans based on individual user profiles, utilizing Body Mass Index (BMI) for customized recommendations. The system ensures that suggestions align with users' health needs and goals, offering comprehensive guidance on healthy eating habits and effective workout routines. It tailors recommendations according to specific BMI categories, promoting healthier lifestyle choices and supporting overall well-being.
